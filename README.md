## GED_KOREA

고졸검정고시 모의고사 0.06V 웹페이지입니다.

## GitHub 에서 설치 방법
1. 고졸검정고시 모의고사 0.05V.html 을 찾아 클릭한다
2. 오른쪽 다운로드 버튼을 누른다 
3. 설치 후 더블 클릭하면 크롬,웹브라우저 에서 나온다
4. 문제가 발견시 즉시 디스코드에 문의를 넣는다
